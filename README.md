# Gpx2hash

Simple Ruby library that converts a GPX 1.1 file into a Hash.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'gpx2hash'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gpx2hash

## Usage

	require 'gpx2hash'

	# Read in GPX file.
	Gpx2hash.readFile "file.gpx"
